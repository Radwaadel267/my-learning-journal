# AI end to end:

## foundations → modeling

math/ML basics → DL basics → NLP/CV/Audio/Multimodal & their specific tasks [modeling]
model training & optimization: fine-tuning (SFT, LoRA/PEFT), alignment (RLHF, DPO), distillation, quantization [modeling]

## LLM application layer → modeling & systems

LLMs → prompt engineering & context engineering → embeddings & vector DBs → RAG (chunking, hybrid search, reranking, evaluation) → advanced RAG (graph RAG, multi-hop, agentic retrieval) → tool use / function calling → Agentic AI (planning, memory, multi-agent orchestration) [modeling & AI systems engineering]

## productionization → systems engineering

production deployment (serving, inference optimization, batching, caching) → cloud (GPU infra, managed model services) → platform architecture (observability, evals, guardrails, cost/latency tradeoffs, MLOps/LLMOps) → governance (safety, security, privacy, compliance, red-teaming) [AI systems engineering]

## day-to-day leverage → using AI tools

AI-assisted coding (Claude Code, Cursor, Copilot) → prompt libraries & personal workflows → workflow automation (n8n, Zapier AI, scripting with APIs) → domain tools (writing, design, research, data analysis) → building personal agents via APIs + MCP → staying current (papers, newsletters, eval leaderboards, shipping side projects) [using AI tools]
